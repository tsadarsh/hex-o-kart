# hex-o-kart
Team Rudra's testing robot :robot:

Pin reference for ESP8266:

|GPIO|Pin name|Function|
|---|---------|--------|
|2|D4|PWM for Motor A|
|14|D5|PWM for Motor B|
|4|D2|Motor direction 1|
|0|D3|Motor direction 2|
|15|D8|Encoder A1|
|5|D1|Encoder A2|
|12|D6|Encoder B1|
|13|D7|Encoder B2|

Pin reference fir ESP32:

|GPIO|Pin name|Function|
|---|---------|--------|
|12|D12|Motor A1|
|13|D13|Motor A2|
|25|D25|Motor B1|
|26|D26|Motor B2|
|21|D21|Encoder A1|
|19|D19|Encoder A2|
|34|D34|Encoder B1|
|35|D35|Encoder B2|
